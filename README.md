# GridWay Proxy Manager

GridWay — это универсальное решение для управления VLESS-прокси. Программа позволяет легко переключаться между серверами, балансировать нагрузку и настраивать правила маршрутизации.

## Особенности
* **Native GUI:** Удобный графический интерфейс для Windows.
* **Web-Control:** Управляйте серверами через браузер по адресу `http://localhost:2081`.
* **Smart Routing:** Исключение доменов и IP-адресов из прокси (Direct) прямо из интерфейса.
* **Balancer:** Автоматический выбор самого быстрого сервера на основе TCP Ping.
* **Portable:** Не требует установки, работает сразу после запуска.

## Как начать работу
1. Перейдите в раздел [Releases](https://github.com/misha05shestak-ops/GridWay-Proxy-Manager/releases).
2. Скачайте нужную версию (`GridWayProxy.exe` для Windows или `GridWay_Linux` для Linux).
3. Запустите файл.
4. Откройте браузер и перейдите по адресу `http://localhost:2081` для настройки.

## Настройка
* Логин по умолчанию: `admin`
* Пароль по умолчанию: `admin`
* Рекомендуется сменить пароль во вкладке "Настройки" после первого входа.

## Требования
* Windows 10/11 или Linux (x64).
* Наличие установленного ядра [Xray-core](https://github.com/XTLS/Xray-core) в одной папке с программой.
