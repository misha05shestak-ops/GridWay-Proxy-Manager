# GridWay-Proxy-Manager
Мощный инструмент для управления прокси-соединениями с графическим интерфейсом и веб-панелью
GridWay — это удобное решение для управления вашими VLESS-серверами. Инструмент поддерживает балансировку трафика, настройку маршрутизации (Direct/Proxy) и предоставляет удобную веб-панель для управления серверами.

Основные возможности:
Графический интерфейс (GUI): Нативное приложение для Windows с поддержкой темных тем.

Веб-панель: Управляйте конфигурациями через браузер на любом устройстве.

Маршрутизация: Гибкие правила для исключения доменов и IP-адресов из VPN.

Балансировщик: Автоматический выбор самого быстрого сервера на основе Ping.

Установка:
Скачайте последнюю версию из раздела Releases.

Запустите .exe файл (для Windows).

Откройте http://localhost:2081 в вашем браузере для доступа к панели управления.

Разработано для эффективного управления сетевым трафиком.
